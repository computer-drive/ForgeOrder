# Server Status 
`server_status`是表示服务器的运行状态的全局变量。本文档讲述不同值代表的含义。

1. `-1`
默认值，服务器未初始化。

2. `100`
Flask初始化完毕，开始运行。但此时未完全启动服务。
以下以`1`开头的值表示服务器已启动。

3. `101`
连接到日志数据库，初始化日志记录功能。

4. `200`
Flask初始化且启动所有服务。服务器正常运行。

5. `299`
服务器正在退出。

6. `300`
服务器在启动过程中出错。

7. `400`
服务器在运行过程中出错。


# Server Info
`server_info`是表示服务器在当前状态下的详细信息的全局变量。一般来讲，在服务器运行出错时，`server_info`才为非空字符串。
